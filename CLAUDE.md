This is a Hebrew math bagrut summary website for 11th grade 5-unit students. Stack: Next.js 16, TypeScript, Tailwind CSS v4, KaTeX for math formulas. Topics: motion problems, sequences, probability, geometry, trigonometry, trigonometric functions, root functions, optimization problems. All content must be in Hebrew with RTL support. Main content file is data/topics.ts. Run with npm run dev, available at localhost:3000.
